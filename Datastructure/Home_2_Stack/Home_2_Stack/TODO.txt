()+-*/와 같은 특수문자가 포함됨 -> stack 내부자료를 char형 배열로 표현

대략적 모델
String: 문자형으로 저장, integer: 정수는 정수형으로 저장
1. 입력받음 (String)

2. initStack 실행 - 스택 초기화

3. 후위연산식(정수형 스택 사용)
우선순위 비교(checkPriority) -> 크면 push 작거나 같으면 pop


4. calc 실행 - 실제 연산 수행
숫자는 정수화해 처리

Operator 크기 ( * / + - ) 순 -> Stack에 넣을때

reference
http://programmerchoo.tistory.com/9

식
5*(15+(2*5-3)*(54/2))*123-30*50/1500

후위식
5 15 2 5 * 3 - 54 2 / * + * 123 * 30 50 * 1500 /  -

결과 125459

123-30*50/1500
= 122

5*(15+(2*5-3))	->	5 15 2 5 * 3 - + *
= 110

5*(15+2*5-3) -> 5 15 2 5 * + 3 - *
= 110
